# 📚 AI Study Assistant

An AI-powered learning platform that helps students turn their notes into interactive quizzes and study materials.

## 🚀 Features

📸 Upload or capture up to 10 pages of handwritten or printed notes

🔍 OCR-powered text extraction from images

🤖 AI-generated study materials including:

* ✅ Multiple Choice Questions (MCQs)
* ✍️ Short Answer Questions
* 📖 Long Answer Questions
* ✔️ True / False Questions
* 📝 Fill in the Blanks
* 🎤 Viva / Interview Questions
* 🧠 Higher-Order Thinking Questions (Analyze, Compare, Evaluate, Why)

📚 Automatic note summarization

🎯 Interactive quiz mode

🏆 Smart scoring system

* Correct Answer = +10 Points
* Wrong Answer = -2 Points
* Skipped Question = 0 Points

📈 Performance tracking and learning progress

📄 Auto-generated Certificate of Completion (Demo)

## 🎯 Goal

To make studying more engaging by automatically converting notes into personalized quizzes, helping students practice, identify weak areas, and improve retention through active learning.
